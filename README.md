# Temperature Converter

A lightweight, interactive web application that converts temperatures across different units in real-time. This project focuses on a clean user interface and precise mathematical conversions.

## Features

*   **Real-time Conversion**: No "Calculate" button needed—results update instantly as you type.
*   **Multiple Units**: Supports **Celsius**, **Fahrenheit**.
*   **Error Prevention**: It prevents the application from trying to perform math on a null state.

## Built With

*   **HTML**: Semantic structure for the converter interface.
*   **CSS**: Custom styling with a focus on modern UI/UX and layout centering.
*   **JavaScript**: Handles the logic, event listeners, and mathematical formulas.

## Formulas Used

The application utilizes the following conversion logic:

*   **Celsius to Fahrenheit**: $F = (C \times 9/5) + 32$
*   **Fahrenheit to Celsius**: $C = (F - 32) \times 5/9$


## Getting Started

  **Open the project**:
    Simply double-click the `index.html` file to launch the application in your preferred browser.
